## Why Rustalink?

<CardGrid stagger>
  <Card title="Blazing Fast" icon="rocket">
    Built with Rust and Tokio for minimal overhead and maximum throughput.
  </Card>
  <Card title="Low Latency" icon="bolt">
    Deterministic memory management ensures consistent audio delivery without GC pauses.
  </Card>
  <Card title="Feature Rich" icon="setting">
    Native support for Equalizer, Bassboost, Timescale, and many more audio filters.
  </Card>
  <Card title="Developer Friendly" icon="document">
    Comprehensive REST and WebSocket API with multi-language client support.
  </Card>
</CardGrid>
